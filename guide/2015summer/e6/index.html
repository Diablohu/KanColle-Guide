海域攻略简述

  • 联合舰队出击，可自由选择航母机动部队或水上打击部队编制
  • 锁船：与E3共享锁船，可出击E7决战
  • 简易航向规则
    • 水上打击部队 -> A
    • 航母机动部队，当存在以下组合时会驶向A，否则驶向C
      • 夕立+绫波
      • 吹雪+古鹰
      • 比睿+雾岛
  • 除地图上标注之外，其他需要注意的战斗点
    • B：敌水雷战队
    • D：夜战，敌配置有——战舰RU级改Flagship，战舰RI级改Flagship
  • Boss阵容
    • 空母栖姬 (新型) + 战舰栖姬 + 轻巡TSU级Elite + 驱逐NI级后期型Elite + 驱逐NI级后期型Elite + 驱逐NI级后期型Elite
    • 最终阶段：空母栖姬 (新型) + 战舰栖姬 + 轻巡TSU级Elite + 轻巡TSU级Elite + 驱逐NI级后期型Elite + 驱逐NI级后期型Elite
  • 目前建议选择航母机动部队，走上路或利用带路条件走下路。装备方面，强烈建议搭载满级烈风，除了最大格放攻击编队、保证一队彩云外，其他所有格全部搭载烈风

  • 马鹿油 (Boss)
