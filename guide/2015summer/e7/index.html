海域特殊机制

本最终决战有着全新的特殊机制

Boss本体有着超高的装甲值 (甲难度333点)，击杀海域内某些战斗点的旗舰可将其削弱，虽然显示数据不变，但实际上确实有降低效果。该效果会在每天的日本时间零时 (北京时间23:00) 清空重置。

击杀旗舰次数对Boss装甲值的影响目前不明，联合舰队旗舰建议击杀数次以保证削甲效果。(个人全部击杀了3次)

“削甲作战”

联合舰队旗舰在此建议，在正式出击之前，先削弱Boss的装甲值

  • 对于 X（港湾栖姬）和 Y（离岛栖鬼），可采用同一套阵容：水上打击部队，舰队中加入 秋津丸 (あきつ丸)，2队配置多CA装备三式弹，1队BB装备穿甲弹
    • 如果没有秋津丸也不必担心，水上打击部队驶入 Y 的几率是很高的
  • 对于 K（航空战），可采用航母机动部队，1队用满CV名额，制空配到300+ (外加全部7级舰战)，其他所有空格全部摆满攻击机，1队也无需配备BB，2队夜战重视，但无需CI，全部夜战二连即可

正式出击

  • 联合舰队出击
  • 锁船：参与E5作战的舰娘无法出击该最终决战
  • 简易航向规则
    • 目前推断全地图航向随机
    • 航母机动部队：高几率出击时驶向E，高几率J->M
    • 水上打击部队：高几率出击时驶向B
  • 除地图上标注之外，其他需要注意的战斗点
    • C：敌补给运输舰队
    • J：空母WO级改Flagship(新型)
    • L：战舰栖姬 + 2x 战舰RU级Flagship
    • M：2x 空母WO级改Flagship(新型) + 2x 轻巡TSU级Elite
  • 本海域限制SS和/或CLT出战，有SS和/或CLT的舰队将无法抵达Boss点
  • Boss阵容
    • 防空栖姬 + 战舰栖姬 + 重巡NE级Elite + 重巡NE级Elite + 驱逐NI级后期型Elite + 驱逐NI级后期型Elite
    • 最终阶段：防空栖姬 + 战舰栖姬 + 战舰栖姬 + 重巡NE级Elite + 驱逐NI级后期型Elite + 驱逐NI级后期型Elite
  • 建议先进行削甲 磨血/通常阶段，水上打击与机动部队都可以使用 斩杀/最终阶段，建议配置航母机动部队，期待航向 E -> G -> J -> M -> Boss，虽然随机但几率较大 M点的敌制空战力极高，如果拿不到制空确保也不要勉强，保证制空优势即可 2队必须夜战重视，同时要配备多夜战CI

其他细节

  • Boss战时不要期待航母空袭的效果，Boss本体有着390点对空属性
  • 强烈建议配备道中支援，可有效提高道中通过率
  • 如果有条件，请将阿武隈升级到改二，由于SS和CLT被限制，只有阿武隈改二可以先手雷击
  • 如果有条件，请在2队配置俾斯麦drei，她在2队的先手炮击有很高的几率可以击沉敌小船，或将敌航母中破
  • 对于航母机动部队，建议将BB放在下方位置(5、6号位)，以期其有更高的几率攻击到Boss本体
  • 在2队，除了旗舰位的夜战CI外，建议将夜战二连的舰娘放置在靠上的位置，其余夜战CI配置的舰娘配置在靠下的位置，这样即便二连打到Boss本体，即便擦弹也不会是极低伤害
