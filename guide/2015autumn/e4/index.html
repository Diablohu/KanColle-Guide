E4 - 西方戦線！機動部隊派遣

通关奖励

  • 甲：勋章 x1，熟练搭乘员 x2，改修资材 x8，洋上补给，Fw190T改
  • 乙：勋章 x1，熟练搭乘员 x1，改修资材 x4，洋上补给，Bf109T改
  • 丙：勋章 x1，熟练搭乘员 x1

海域目标

击破Boss，最终（血条已空）战时必须击沉Boss

  • 甲、乙：至少击沉Boss 5次
  • 丙：至少击沉Boss 4次

海域要点简述

  • 联合舰队出击
  • 锁船：参与该战斗的舰娘可出击E5，无法出击E1、E2、E3
  • 除地图上标注之外，其他需要注意的战斗点
    • 海域内的潜艇点都有几率出现梯形阵
    • I存在空母WO级Flagship改（新型）
    • 道中航母战大约有120左右的制空战力
  • 部分航向规则
    • 航母机动部队驶向 A，但之后的航线目前仍未判明规则
    • 水上打击部队驶向 D
  • Boss存在削甲机制，如果感觉击杀Boss有困难可以采用
    • 分别击沉J和M的旗舰各一次可弱化Boss潜水栖姬的装甲值
    • 只有航母机动部队才会看到Boss立绘变化，水上打击不对看不到，有可能是Bug。但削甲效果是存在的
    • 目前不清楚每日日本时间05:00是否会重置装甲值
  • Boss阵容
    • Boss本体为 潜水栖姬
    • 如果出现空母WO级Flagship改（新型）大约有130点制空战力
    • 通常阵容：潜水栖姬 + 2x 潜水SO级 + 空母WO级Flagship改（新型） + 2x 驱逐HA级后期型Elite
    • 通常阵容：潜水栖姬 + 2x 潜水SO级 + 战舰栖姬 + 2x 驱逐HA级后期型Elite
    • 最终阶段：潜水栖姬（强化） + 潜水SO级 + 战舰栖姬 + 空母WO级Flagship改（新型） + 2x 驱逐HA级后期型Elite

一句话攻略

采用水上打击部队，一来机动部队目前航向规则不明，二来水上部队可以有效的保护2队反潜力量。除了保证一定数量的CV/CVL和BB外，派出所有可用的对潜力量包括，高等级DD、CL、4格CL和练巡。2队可加入多个CLT，提高道中通过率。推荐对反潜装备进行改修。推荐配置道中支援。如果仍有困难可考虑对Boss进行削甲。
