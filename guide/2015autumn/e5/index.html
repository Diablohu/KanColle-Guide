E5 - 乗り越えろ！バニラ湾夜戦！

通关奖励

  • 甲：驱逐舰 萩风，勋章 x2，甲种勋章
  • 乙：驱逐舰 萩风，勋章 x2
  • 丙：驱逐舰 萩风，勋章 x1

海域目标

本海域有2个连续的作战阶段，分别为运输作战和攻坚作战。

运输作战，目标TP：

  • 甲：360
    • 单次获取60点可6次通关（13桶A胜为60点）
    • 单次获取52点可7次通关（12桶A胜为57点）
  • 乙：？
  • 丙：？

攻坚作战击破Boss，最终（血条已空）战时必须击沉Boss

  • 甲：至少击沉Boss 5次
  • 乙：至少击沉Boss ？次
  • 丙：至少击沉Boss 3次

海域要点简述

  • 单舰队出击
  • 阵容必须为 1CL + 4DD + 1任意巡洋舰或DD
  • 不锁船：任何舰娘都可以参与该海域的战斗，参与战斗的舰娘也不会被锁定
  • 除地图上标注之外，其他需要注意的战斗点
    • JL的夜战敌人配置较为强力，建议装配夜战装备
    • 海域中所有战斗敌人都没有制空战力
  • 当舰队中存在川内、时雨、江风时会满足完全带路条件，固定 B -> E 或 C -> G
    • 不完全满足时有一定随机性
    • 完全不满足时固定最远航线
  • Boss存在削甲机制，如果感觉击杀Boss有困难可以采用
    • 削甲仅针对攻坚作战，在运输作战时削甲不起作用
    • 击沉20个[[PT小鬼群]]即可达成削甲目标，H 固定刷新4个，可以来这里进行削甲
    • 削甲成功后Boss立绘发生变化
    • 日本时间每日05:00装甲值重置
  • Boss阵容
    • Boss本体为 驱逐水鬼
    • 通常阵容：驱逐水鬼 + 驱逐HA级后期型Elite + 2x 驱逐RO级后期型Elite + 2x 驱逐I级后期型Elite
    • 通常阵容：驱逐水鬼 + 驱逐HA级后期型Elite + 2x 驱逐RO级后期型 + 2x [[PT小鬼群]]
    • 运输作战-最终阶段：驱逐水鬼 + 驱逐HA级后期型Elite + 2x 驱逐RO级后期型 + 2x [[PT小鬼群]]
    • 攻坚作战-最终阶段：驱逐水鬼（强化，可开幕雷击） + 驱逐HA级后期型Elite + 2x 驱逐RO级后期型Elite + 2x 驱逐I级后期型Elite

运输作战 - 一句话攻略

推荐剩余的一个自由位选用 CLT、CAV 或 阿武隈改二，均可提供一定的开幕攻击能力，有效提高道中通过率。如果满足带路条件，道中支援意义不大，因为道中支援只会影响第一战。建议配置决战支援，毕竟装备了很多运输桶损失战力。

攻坚作战 - 一句话攻略

注意本阶段需要一定的索敌能力才可进入Boss战，但要求不高，装备一个小型电探即可达标。如果感觉击杀Boss有压力可选择对其进行削甲，由于每日重置，削甲前请安做好时间安排。削甲成功后Boss立绘发生变化，效果也很明显，甚至不需要决战支援就可以完成对Boss的击杀。

需要注意的掉落

  • NEW 岚 (Boss)
  • 野分 (J/L/Boss)
  • 江风 (J/L/Boss)
  • 阿贺野 (Boss)
  • 能代 (Boss)
  • 矢矧 (Boss)
  • 酒匂 (Boss)
  • 三隈 (Boss)
  • まるゆ (Boss)
