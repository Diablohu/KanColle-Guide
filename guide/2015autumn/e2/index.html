E2 - コロネハイカラ沖海戦

通关奖励

  • 甲：应急修理女神 x1，给粮舰 间宫 x1，战斗粮食 x2，补强增设 x2，勋章 x1
  • 乙：应急修理女神 x1，给粮舰 间宫 x1，战斗粮食 x2，补强增设 x1
  • 丙：应急修理女神 x1，给粮舰 间宫 x1，战斗粮食 x1

海域攻略简述

  • 单舰队出击
  • 阵容必须为 1CL + 5DD，且CL旗舰
  • 锁船：参与该战斗的舰娘可出击E1、E3、E5，无法出击E4
  • 本关采用全新的游戏机制：运输作战。关于该机制请阅读活动须知中的相关内容。
  • 除地图上标注之外，其他需要注意的战斗点
    • 虽然同为夜战点，但F中的敌人均为单发攻击配置，而H中的敌人二连和CI攻击并存，显然上路的夜战更为凶险。下路夜战点G存在新敌人PT小鬼群
  • 当舰队中存在神通、滨风、皋月、三日月、雪风其中的一名舰娘时，会驶向A
    • 当以上5名舰娘全部存在时固定驶向D，否则随机航向
  • Boss阵容
    • 由于本海域目标为运输作战，在攻略期间不需要击杀Boss本体，只需要获得Boss战胜利即可
    • Boss本体为 轻巡栖姬
    • 通常阵容：轻巡栖姬 + 轻巡HE级Flagship + 轻巡HO级Flagship + 驱逐RO级后期型 + 2x 驱逐I级后期型
    • 最终阶段：轻巡栖姬（强化） + 轻巡HE级Flagship + 轻巡HO级Flagship + 驱逐RO级后期型Elite + 2x 驱逐I级后期型
  • 本海域的攻略难点其实在带路条件上。如果你的舰队可以满足带路条件，那可以采用一个非常精妙的策略：所有舰娘全部装备运输桶，同时配置决战支援，采用航母空袭模式，即2驱逐+4航母，航母挂舰攻和舰爆（感谢NGA网友ddj_1mol提供该打法）

  • 天津风 (Boss)
