E2 - 出撃！「礼号作戦」

通关奖励

  • 甲：驱逐舰 初月，改修资材 x8，洋上补给 x2，熟练搭乘员 x1，勋章 x1
  • 乙：驱逐舰 初月，改修资材 x5
  • 丙：驱逐舰 初月

海域目标

击破Boss，最终（血条已空）战时必须击沉Boss

  • 甲：至少击沉Boss 6次
  • 乙：至少击沉Boss 6次
  • 丙：至少击沉Boss ？次

海域要点简述

  • 单舰队出击
  • 不锁船
  • 除地图上标注之外，其他需要注意的战斗点
    • G：战舰TA级Flagship旗舰的敌强力水面舰队
    • M、N：大量PT小鬼群
  • 出击舰队配置直接决定出击位置，目前猜测满足以下条件时从北线出击，其他情况一律南线出击：
    • DD 至少 3
    • 没有 BB/BBV/CV/CVL/SS
    • CA、CAV、CL 均不超过 2
  • 北线作战
    • 当满足以下条件之一时，A后会驶向E
      • 如果舰队中包含 霞、清霜、朝霜、足柄 或 大淀 其中至少 4 只舰娘。（部分符合条件时随机航向）
      • 舰种满足 4DD + 1CA + 1CL 或 3DD + 1CA + 1CL + 1CLT
    • 如果你没有火箭弹储备，DD的战斗力会受到严重制约——但可以用3DD3CA/CAV的阵容解决这个问题。
    • 如果基本没有礼号作战相应的舰娘，或者没有固定舰种组合的配置，B之后的线路上战斗数量众多，此情况下不建议进行北线作战。
    • 道中敌人除了航空战外没有空中战力，Boss战有一定的空中战力，建议配置对空力量。
  • 南线作战
    • 该线作战可配置航母，请配置至少150点的制空战力
    • 当同时满足以下条件时可规避D
      • DD 至少 2
      • CV+CVL 最多 1
      • 不包含 大和级BB
      • 霞旗舰时 BB+BBV+CV+CVL 最多 3
      • 其他旗舰时 BB+BBV+CV+CVL 最多 2
      • 以上来自日文WIKI的分析
  • 感谢所有回复评论的朋友测试出的航向规则！
  • Boss本体为 集积地栖姬，路基单位，注意配置三式弹和对地火箭弹。Boss的装甲值较低，DD、CL装备火箭弹也可以有效杀伤。

一句话攻略

根据自身的舰娘和装备储备情况决定作战线路。北线作战需保证一定的对空力量，南线作战需保证较强大的正面火力。无论哪条线路都需要配置三式弹和对地火箭弹。

北线作战阵容建议

  • 如果你有礼号作战相关舰娘且等级较高，可采用以下配置：
    • 3DD + 1CA + 2CAV
    • 3DD + 2CA + 1CAV
    • 3DD + 1CL + 1CA + 1CAV
  • 如果没有礼号作战相关舰娘，也可以考虑北线，使用固定舰种的组合：
    • 3DD + 1CA + 1CL + 1CLT

对路基Boss杀伤力方面，除了传统的三式弹外，此次作战火箭弹也有很有效的杀伤力，装备多枚火箭弹可以进一步提高杀伤力——大淀表示她很强！
